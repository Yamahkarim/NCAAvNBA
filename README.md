# NCAA vs NBA 
![bball](img/bball.jpg) 
### How much better are the pros?

## Objective
Gather, analyze, and visualize data from both the NCAA and NBA to quantify the difference between Professional and Amateur basketball player. Our results hope to shed light onto the importance of superstar talent within both leagues. 

## Technoligies 
 
- Python 3 
- Matplotlib 
- Pandas 
- Jupyter Notebook 
- JSON 

## Explore 

1.) Clone Repo 

2.) Create and initialize a new [virtual enviornment](https://uoa-eresearch.github.io/eresearch-cookbook/recipe/2014/11/20/conda/)

3.) Download [Python 3](https://www.python.org/downloads/)

4.) Install [Jupyter Notebook](https://jupyter.readthedocs.io/en/latest/install.html), [matplotlib](https://matplotlib.org/users/installing.html), [Pandas](https://pandas.pydata.org/pandas-docs/stable/install.html) and [Numpy](https://docs.scipy.org/doc/numpy/user/install.html) 

5.) To work through code, navigate to cloned repo within a CLI and initialize a jupyter notebook with the command `jupyter notebook` 

6.) The repo contains all raw data as well as the projet proposal and presentaion.
